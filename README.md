# Heart Animation ❤️

A beautiful animated heart particle effect created with HTML5 Canvas and JavaScript.

## Live Site

Visit the deployed site: https://mayankgotnolife.github.io/birthday-heart/

## Description

This project features an animated heart that emits particle effects. The heart is drawn using mathematical equations and particles are animated using canvas.

## Original Source

A Pen created on CodePen.

Original URL: [https://codepen.io/Facundo-Sencar/pen/emYajdo](https://codepen.io/Facundo-Sencar/pen/emYajdo).

## Files

- `index.html` - Main HTML file (deployment version)
- `script.js` - JavaScript animation logic
- `style.css` - Stylesheet
- `heart/Heart/src/` - Source files
- `heart/Heart/dist/` - Distribution/build files

## Usage

Simply open `index.html` in a web browser to see the animation.
